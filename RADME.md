# HRMS / AdScale One ERP

Enterprise agency operating system built phase by phase.

## First Version Stack

- Frontend: React / Next.js + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Auth: JWT + Bcrypt
- Realtime later: Socket.IO
- Queue/Cache later: Redis
